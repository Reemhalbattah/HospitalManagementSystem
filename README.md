# Hospital Management System (JavaFX)

A Hospital Management System developed as a university project using Java and JavaFX. The project simulates a hospital environment by managing patients, doctors, appointments, and medical records through a graphical user interface.

---

## Features

- Patient registration
- Doctor management
- Appointment scheduling
- Medical records management
- Graphical User Interface (GUI)
- Object-Oriented Programming (OOP)

---

## Technologies Used

- Java
- JavaFX
- NetBeans IDE
- Object-Oriented Programming (OOP)

---

## Learning Outcomes

This project demonstrates:

- Java programming
- JavaFX GUI development
- Object-Oriented Programming (OOP)
- Event handling
- Software design principles
